# TODOLISTCPP

cli task runner and editor

### Usage

* configuration
```
make configure
```

* build 
```
make build
```

* run
```
out/bin/todocpp
```

* build on change
```
make watch
```

* unit tests
```
make tests
```

### Pre-requisites

* [cmake](https://cmake.org)
* [gyp](https://gyp.gsrc.io)
* [ninja build system](https://ninja-build.org)
